# CustomKaliSetup
To automatically download &amp; install most used pentest tools on Kali or any Linux distrib.

### Download the gits and scripts
```
> ./download.sh
```

### Install the tools (create venv, alias, ...)
```
> ./install.sh
```

### Update the gits
```
> ./update.sh
```

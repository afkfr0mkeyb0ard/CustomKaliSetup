# CustomKaliSetup
To automatically download &amp; install most used pentest tools on Kali or any Linux distrib.

### Setup
```
git clone https://github.com/afkfr0mkeyb0ard/CustomKaliSetup.git
cd CustomKaliSetup
chmod +x setup.sh
./setup.sh
```
